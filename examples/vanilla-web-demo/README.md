# Rajeev SDK â€” Vanilla Web Demo

A **zero-dependency** single-page demo using plain HTML, CSS, and vanilla JavaScript.

## Getting Started

1. Open `index.html` in any modern browser (double-click the file).
2. Click the library tabs to switch between demos.
3. Press action buttons to see simulated SDK calls in the output log.

**No build step. No npm. No framework.**

## Libraries Demonstrated

All 15 Rajeev SDK libraries: Vault, Network, Locale, Sync, Notify, UI Tokens,
Auth, Payments, Camera, Deep Link, Document, Edge AI, Media, Video Editor, and App Shell.
